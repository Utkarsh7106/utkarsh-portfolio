# Utkarsh Mishra — Portfolio Website

Personal developer portfolio built from scratch using vanilla HTML, CSS, and JavaScript. Currently on **V3**.

**Live Site →** [utkarsh7106.netlify.app](https://utkarsh7106.netlify.app)

---

## About

This is my personal portfolio website showcasing my projects, skills, and experience as a final-year Computer Engineering student at Ramrao Adik Institute of Technology, Mumbai. The site is designed with a warm paper aesthetic, dark mode support, and smooth scroll animations.

---

## Features

- **Dark / Light mode** — toggle with persistent `localStorage` preference
- **Scroll animations** — elements reveal on scroll via `IntersectionObserver`
- **Project filter tabs** — filter projects by category (Academic / Internship / Personal)
- **Responsive design** — works across desktop and mobile
- **Noise texture overlay** — subtle grain effect for depth
- **Animated progress bars** — QSpiders training progress, driven by a JS config object
- **Resume link** — opens in a new tab via Google Drive preview

---

## Tech Stack

Built with no frameworks or dependencies — pure vanilla web.

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- JavaScript (ES6+, IntersectionObserver, localStorage)
- Google Fonts — Cormorant Garamond, Outfit, DM Mono

---

## Local Development

No build tools or package managers needed. Just open `index.html` in your browser.

For live reload during development, use the VS Code **Live Server** extension.
